This repository contains the official code implementation and supplementary materials of the paper "CEBW-YOLO11: A YOLO11-Based Model for Lithium Battery Electrode Sheet Defect Detection". This model, based on YOLO11n, significantly improves the detection accuracy and robustness of surface defects (agglomeration, bubbles, cracks, scratches) of lithium battery electrode sheets by integrating four modules: C3k2-DWR, EAC attention compression, BiFPN-DySample, and Wise-Inner-IoU.
